# Getting Started on Ubuntu 20.04.5 LTS 
- Install `unzip`
- Install [NVM](https://nodejs.org/en/download/)
- Install and select Node 18+ using: `nvm ls-remote; nvm install <VERSION>; nvm use <VERSION>` 
- Run `npm install`
- Run `npm run serve`
- Open `http://localhost:8080/`
